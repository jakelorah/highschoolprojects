I.          COURSE DESCRIPTION :

 

Computer Science I introduces students to both the internet and applications programs. 
The primary focus is on the internet, Hypertext Markup Language (HTML) and Microsoft
Access.  The aim is to teach a disciplined design style that produces professional and
well-documented web sites and data bases.

 

Students will learn the basics of how the World Wide Web operates and how to code and
create web documents.  Anchors and hyperlinks are learned that allow site navigation. 
Background images and color is discussed.  Following this, tables are learned that enable
page layouts to be manipulated.  Frames and Forms are learned.  Frames provide additional
layout capabilities and forms necessary for user input.   

 

Access is the database management system in Microsoft Office.  Students will learn the basics
of how a database works as well as how it is designed, created and managed. 

 

The course includes applications programming (Microsoft Word, Excel and Powerpoint) to
complement Microsoft Access. 

 

Students will provide support, maintenance and development for the PCTI school website.  

           

 

 

 

            Teaching time frame :           1 year

 

            Suggested grade level :       9

 

Prerequisite :                         Acceptance to AOIT

IV.       INSTRUCTIONAL STRATEGIES :

 

Computer Science I encompasses many different types of open-ended problems of a
mathematical or business nature.  The goal of the Computer Science I class is to reinforce
existing skills and develop new skills in various internet based software.

 

Microsoft Access is further developed and is used in conjunction with ASP and VBScript to
solve various business related problems. 

 

            Therefore, all students must demonstrate the following skills :

 

                        1.         Critical thinking

                        2.         Decision making

                        3.         Software engineering

                        4.         Use of technologies

                        5.         Self-management skills

                        6.         Time-management skills

                        7.         Teamwork

 

In order to apply these skills, the instructional strategies will incorporate solving a number
of case studies.  By analyzing these open-ended problems, students apply all of the above
skills in great detail.  They create a model solution by applying the divide and conquer technique.

V.        COURSE OUTLINE

 

1.         HTML / Dynamic HTML and Web Page design

A.        Designing a Web Page 

a.         Learn how HTML handles color

b.         Create a color scheme for a web page

c.         Insert a background image into a web page

d.         Create spot color

e.         Learn about different image formats

f.          Learn to control placement and appearance of images on a  web page

g.         Work with client-side image maps

B.         Design a Web Page with Tables

a.         Create a text table

b.         Create a graphical table

c.         Create table headers

d.         Control the appearance of a table and table text

e.         Create table cells and spans

f.          Use nested tables to enhance page design

C.        Using Frames in a Web Site

a.         Create frames for a web site

b.         Control the appearance and placement of frames

c.         Control the behavior of hyperlinks

d.         Use magic target names to specify the target for a hypetext link

D.        Create Web Page Form

a.         Learn about CGI scripts

b.         Create form elements

c.         Create hidden fields on a form

d.         Work with form properties

e.         Learn how to send data to a CGI script

 

2.         Database (Microsoft Access)

A.                 Creating a database

a.         Learn what a database is and what Access is

b.            Learn what a data base management system is

c.             Learn data base objects

d.            Create a table in datasheet view

e.             Create a primary key

f.              Use date/time and currency data types

g.             Enter records

h.             Change column width

B.                 Database Forms

a.         Create a form from an existing datasheet

b.                  Learn form design view

 

 

 

 

 

C.                 Editing and Printing

a.         Add, delete and move fields in design view

b.                  Add, delete and move fields in datasheet view

c.                   Print, print setup, headers and footers

d.                  Hide fields

e.                   Learn to edit a record

f.                    Add and delete a record

g.                   Learn to insert gridlines and form backgrounds

D.                 Searching an Sorting

a.         Find records

b.                  Replace data

c.                   Search using wildcards

d.                  Sort records

e.                   Quick sort and Multiple sort

f.                    Filter, sort and edit a record subset

E.                  Queries

a.         Create a query

b.                  Change and save a query design

c.                   Learn to use fields and reposition fields for a query

d.                  Renaming and printing a query

e.                   Learn to change field format and names

f.                    Create query using multiple tables

g.                   Print query table

F.                  Reports

a.         Create a report

b.                  Edit report sections

c.                   Create a report from a query

d.                  Learn to insert report sections, controls, group controls and footers

e.                   Learn to insert report statistics

f.                    Learn to create a database with the database wizard

 

            3.         Photoshop

A.        Fundamentals

B.         Basic image editing

C.        Filters

D.        Color and Resolution

E.         Bitmap / Pixel images
