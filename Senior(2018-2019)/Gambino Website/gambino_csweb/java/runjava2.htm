How to Enter a Java Program

 

1.         Open a Notepad file from the start menu.

 

import java.awt.Graphics;

                        import java.applet.Applet;

 

                        public class array1 extends Applet {

                                    int n[ ] = {32,27,64,18,95,14,90,70,60,37};

           

                                    // paint the applet

                                    public void paint( Graphics g )

                                   {

                                                int yPosition=25; // starting y position on applet

                                                g.drawString( "Element",25,yPosition );

                                                g.drawString( "Value",100,yPosition );

                                                for( int i=0;i < n.length;i++ )

                                               {

                                                            yPosition+=15;

                                                            g.drawString( String.valueOf(i),25,yPosition );

                                                            g.drawString( String.valueOf(n[i]),100,yPosition );

                                                }         

                                   }

                        }

 

 

  1.  When you are done typing, go to File-Save As.

        How to Compile a Java Program

 

  1.  Open DOS Cmd command prompt from the Start menu.

 

  1.  At the prompt, type cd c:\ , and the directory your program is in.

 For example, if your program is in C:\c03\LastName\cs3\java   then type:

 

          C:\WINNT> cd c:\c03\LastName\cs3\java            and press enter.

 

    

 

 

At this prompt, type the following:  

                                            

  c:\c03\LastName\cs3\java>   c:\jdk1.3.1_02\bin\javac array1.java

 

     

  1. If there are errors, go back to your program and fix them. If not, then you are ready
    to  run your program. You can run your program by using an Internet browser or
    using Appletviewer.        

 

 

        How to Run a Java Program Using An Internet Browser

 

  1. Open a separate Notepad file.

 

  1. Type in a small HTML file similar to this one:

 

            <html>

            <body>

            <h1>Array Program</h1>

            <applet code="array1.class" width=300 height=250>

            </applet>

            </body>                                                                                              

            </html>                                                                                                                                                                    

 

 

  1. When you are done, go to File-Save As.

 

 

 

 

 

 

 

 


  1. To run the program, open your Internet browser.

 

5.  On the address bar, type in the directory where your HTML file is in. For example, if
